# 出版构建指南（Markdown + Pandoc + Typst）

## 目标

将本书 Markdown 内容一键构建为排版 PDF，形成可重复构建流程。

## 依赖

- `pandoc >= 3`
- `typst >= 0.14`

## 快速开始

```bash
cd /Users/eevv/focus/buildbook
make check
make pdf
```

生成文件：

- `dist/book.typ`
- `dist/book.pdf`

## 目录与模板

- 元数据：`book/build/metadata.yaml`
- 版式主题：`book/theme/book-theme.typ`
- 构建脚本：`scripts/build-book.sh`
- 一键命令：`Makefile`

## 常见调整

1. 书名/副标题/作者：修改 `book/build/metadata.yaml`
2. 页边距/字体/标题样式：修改 `book/theme/book-theme.typ`
3. 章节编译顺序：修改 `scripts/build-book.sh` 的 `INPUTS`

## 说明

- 研究资料目录 `book/research/compare/` 默认不进入正式正文编译。
- 若本机缺少中文字体，可在 `book/theme/book-theme.typ` 调整字体回退列表。
