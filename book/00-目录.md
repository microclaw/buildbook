# 《从零构建MicroClaw》

## 书籍定位

- 中文书名：从零构建MicroClaw
- 英文书名：Building MicroClaw Systems from Scratch
- 目标读者：AI Agent 工程实践者、后端工程师、平台架构师、技术负责人
- 写作目标：从工程现实出发，完整讲清 MicroClaw 的架构、实现、取舍与演进路径
- 阅读方式：由浅入深，先概念，再流程，再代码，再运维
