# 封面设计 Prompt（O'Reilly 风格启发，原创）

## 主 Prompt

请设计一本技术书封面，风格“受 O'Reilly 经典技术书版式启发”，但必须是原创设计，不得使用任何 O'Reilly 商标、Logo、版式复刻或受版权保护的具体视觉元素。

书名：
《从零构建 MicroClaw》

副标题：
多渠道 Agent Runtime 的架构与实践

作者：
（留空，不显示作者名）

视觉要求：
- 专业、克制、技术感强、书架辨识度高。
- 顶部窄信息栏 + 中央主视觉 + 底部色带，结构清晰。
- 主视觉为“机械化微型龙虾（MicroClaw）”黑白线描插画，融合电路与齿轮细节。
- 背景为浅米色纸纹理，不要纯白。
- 配色以墨黑、炭灰为主，强调色用锈橙或铜绿。
- 2:3 竖版，适合技术书封面印刷。

排版要求：
- 书名置于中上区域，字号最大。
- 副标题置于书名下方，小一号。
- 可加小号英文眉题：MICROCLAW。
- 保留充足留白，不要元素拥挤。

风格关键词：
editorial book cover, technical publishing, engraved illustration, monochrome ink drawing, vintage scientific plate, modern minimalist layout, high typographic hierarchy, print-ready

禁止项：
- 禁止任何 O'Reilly 字样、logo、品牌元素。
- 禁止霓虹赛博风、过度发光、廉价 AI 视觉。
- 禁止乱码、错字、低质量字体。

输出：
- 高分辨率封面图，可直接用于 PDF 书籍封面页。

## 参数建议

- 比例：1024x1536（2:3）
- 质量：high
- 先生成 3-4 个版本，再做一次精修
