# 出版构建

## 三种发布格式

| 格式 | 工具 | 输出路径 | 用途 |
|------|------|----------|------|
| HTML | mdBook | `dist/html/` | GitHub Pages 在线阅读（左右结构） |
| PDF | Pandoc + Typst | `dist/book.pdf` | 印刷与离线阅读 |
| EPUB | Pandoc | `dist/book.epub` | 电子书阅读器（Kindle / Apple Books） |

## 依赖安装

```bash
# macOS
brew install pandoc typst
cargo install mdbook

# Ubuntu / CI
sudo apt-get install -y fonts-noto-cjk fonts-noto-cjk-extra
# pandoc, typst, mdbook 通过 GitHub Actions 自动安装
```

## 一键构建

```bash
# 构建全部三种格式
./deploy.sh

# 仅构建某一种格式
./deploy.sh html    # mdBook → HTML
./deploy.sh pdf     # Pandoc → Typst → PDF
./deploy.sh epub    # Pandoc → EPUB

# 清理构建产物
./deploy.sh clean
```

也可以用 Makefile：

```bash
make            # 等同于 ./deploy.sh all
make html       # 仅 HTML
make pdf        # 仅 PDF
make epub       # 仅 EPUB
make serve      # 本地预览（启动 mdBook 开发服务器）
make check      # 检查依赖是否就绪
make clean      # 清理
```

## 本地预览

```bash
make serve
# 浏览器打开 http://localhost:3000
```

mdBook 开发服务器支持热重载，修改 Markdown 文件后自动刷新。

## GitHub Pages 部署

推送到 `main` 分支即自动触发 `.github/workflows/deploy.yml`：

1. 构建 HTML、PDF、EPUB 三种格式
2. 将 PDF 和 EPUB 复制到 HTML 下载目录
3. 部署到 GitHub Pages

需要在 GitHub 仓库的 Settings → Pages 中将 Source 设为 **GitHub Actions**。

## 关键文件

| 文件 | 作用 |
|------|------|
| `book.toml` | mdBook 配置（书名、语言、输出目录） |
| `mdbook-src/SUMMARY.md` | 左侧导航树定义 |
| `mdbook-src/introduction.md` | 首页/前言 |
| `mdbook-src/theme/head.hbs` | HTML 中文排版自定义 CSS |
| `build/metadata.yaml` | PDF/EPUB 元数据 |
| `build/epub.css` | EPUB 中文排版样式 |
| `theme/book-theme.typ` | PDF Typst 排版主题 |
| `deploy.sh` | 一键构建脚本 |
| `.github/workflows/deploy.yml` | CI/CD 自动部署 |

## 目录结构

```
buildbook/
├── book.toml                        # mdBook 配置
├── deploy.sh                        # 一键构建
├── Makefile                         # Make 快捷入口
├── mdbook-src/
│   ├── SUMMARY.md                   # 左侧导航树
│   ├── introduction.md              # 首页
│   ├── theme/head.hbs               # 中文 CSS
│   ├── chapters -> ../chapters      # 符号链接
│   ├── appendices -> ../appendices  # 符号链接
│   └── assets -> ../assets          # 符号链接
├── build/
│   ├── metadata.yaml                # PDF/EPUB 元数据
│   └── epub.css                     # EPUB 样式
├── theme/
│   └── book-theme.typ               # PDF Typst 主题
├── chapters/                        # 18 章源文件
├── appendices/                      # 4 附录源文件
├── assets/figures/                  # SVG 架构图
├── dist/                            # 构建输出（git ignored）
│   ├── html/                        # GitHub Pages
│   │   └── downloads/               # PDF/EPUB 下载
│   ├── book.pdf
│   ├── book.epub
│   └── book.typ
└── .github/workflows/deploy.yml     # CI/CD
```

## 调整指南

- **修改书名/作者**：编辑 `book.toml`（HTML）和 `build/metadata.yaml`（PDF/EPUB）
- **调整章节顺序**：编辑 `mdbook-src/SUMMARY.md` 和 `deploy.sh` 中的 INPUTS 数组
- **自定义 HTML 样式**：编辑 `mdbook-src/theme/head.hbs`
- **自定义 PDF 排版**：编辑 `theme/book-theme.typ`
- **自定义 EPUB 样式**：编辑 `build/epub.css`
- **添加新章节**：在 `chapters/` 下创建文件，更新 `SUMMARY.md` 和 `deploy.sh`
