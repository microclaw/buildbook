# 项目总览

## 项目目标

将本仓库建设为一本工程型技术书的可持续写作项目，主题为：
《剖析与实现 MicroClaw：基于 v0.1.16 源码的多渠道 Agent Runtime 架构、最小实现与生产实践》。

## 目标读者

- 需要构建 Agent 系统的后端与基础设施工程师
- 负责 AI 产品架构的技术负责人
- 希望从“Demo Agent”走向“可运维系统”的实践团队

## 版本基线

- 上游仓库：`microclaw/microclaw`
- 源码基线：`v0.1.16`
- 固定 commit：`95491b787a61a71f43aeb6556c695a3bd1c006ce`
- 仓库快照：<https://github.com/microclaw/microclaw/tree/95491b787a61a71f43aeb6556c695a3bd1c006ce>

## 交付范围

- 主体：三大部分、18 章
- 附录：能力矩阵、源码导读、实施模板、最小实现主线
- 研究资料：6 篇对比文档（OpenClaw/NanoClaw/Moltis/ZeroClaw/NanoBot/NullClaw）

## 不做范围

- 不写泛 AI 科普
- 不写与 MicroClaw 设计无关的工具教程
- 不做大段无证据的观点性评论

## 完成标准

- 每章具备：问题定义、架构决策、实现细节、风险边界、实践清单
- 全书术语统一、章节颗粒统一、图文引用统一
- 支持后续直接进入排版（Typst/Markdown 出版流）
