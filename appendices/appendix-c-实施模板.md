# Appendix C 实施模板

## 使用方式

本附录提供 4 份可直接套用的模板，分别覆盖：

1. 需求澄清
2. 架构评审
3. 发布检查
4. 事故复盘

建议做法：

- 在正式落地前先填模板 1 和模板 2。
- 每次准备上线前填写模板 3。
- 一旦发生生产事故，强制填写模板 4。

这些模板的目的不是增加流程，而是把“本应提前讲清楚的事情”结构化下来。
